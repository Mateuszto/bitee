### Installation and Setup

1. **Clone the repository**
   ```bash
   git clone [repository_address]
   cd bitee
   ```

2. **Install dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Run development server**
   ```bash
   npm start
   ```
   The application will be available at `http://localhost:4200`

4. **Build for production**
   ```bash
   npm run build
   ```
