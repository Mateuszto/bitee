# Bitee

## Getting Started

To get started with the Angular frontend application, follow these steps:

1. **Install Packages**
   ```bash
   cd frontend
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm start
   # or
   ng serve
   ```
   The application will be available at `http://localhost:4200`


## Tech Stack
**Frontend:** Angular | **Backend:** Nest | **Database:** Postgress
